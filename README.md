# 传讯聊天应用

一个现代化的在线聊天应用，支持多种主题和自定义设置。

## 功能特性

- 🎨 多种颜色主题（金色、蓝色、紫色、绿色等）
- 📱 响应式设计，支持移动端
- 💫 流畅的动画效果
- 🔧 丰富的自定义选项

## 本地运行

1. 使用Python启动本地服务器：
```bash
python -m http.server 8080
```

2. 或者在浏览器中直接打开 `index.html`

## 部署到在线平台

### Vercel 部署
1. 访问 [vercel.com](https://vercel.com)
2. 连接GitHub账号
3. 创建新项目
4. 导入此项目文件夹
5. 自动部署完成

### Netlify 部署
1. 访问 [netlify.com](https://netlify.com)
2. 拖拽项目文件夹到部署区域
3. 自动部署完成

### GitHub Pages 部署
1. 创建GitHub仓库
2. 上传项目文件
3. 在仓库设置中启用GitHub Pages
4. 选择main分支作为源
